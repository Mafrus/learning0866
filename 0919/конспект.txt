Установка программы на примере git:
1. Смотрим документацию!!!
    Также полезно поискать инструкции в сети.
2. Скачиваем ПОДХОДЯЩИЙ дистрибутив.
    32 или 64 разрядный и т.д.
3. Запускаем дистрибутив.
2,3 для Linux/MacOS - используем команду установки
4. Добавляем в Path при необходимости. (только для Windows)
5. Проверяем работу программы.

.gitkeep - специальное имя для файла, удерживающего в гите пустую папку
.gitignore - специальное имя для файла, содержащего список файлов и каталогов, которые не должны быть добавлены в гит

git init
ДОБАВЛЯЕМ ФАЙЛЫ
git add .
git commit -am "ВАШ КОММЕНТАРИЙ"
СОЗДАЕМ РЕПО НА ГИТХАБЕ
git remote add origin ВАШЕ_РЕПО_НА_ГИТХАБЕ
git push --set-upstream origin ВАША_ВЕТКА
